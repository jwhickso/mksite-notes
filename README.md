# mksite-notes

## 项目简介

`mksite-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的存放与展示空间，方便整理和共享各种独立制作的网页内容。

## 目录说明

```
mksite-notes/
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放独立 HTML 页面
│   └── *.html
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录用于存放各个独立的 HTML 页面文件，每个页面为一个独立的归档单元。
- `assets/` 目录存放页面共用的样式、脚本和图片资源，方便统一管理。

## 页面归档说明

- 所有归档页面均以独立 HTML 文件形式存放在 `pages/` 目录下。
- 页面内容可包含文本、多媒体、交互组件等，但应保持自包含性，避免依赖外部不可控资源。
- 归档页面不涉及任何商业推广、用户追踪或诱导点击行为。
- 如需添加新页面，请按照现有命名风格在 `pages/` 下创建新的 `.html` 文件，并在 `index.html` 中添加链接。

## 维护说明

- 本仓库由维护者不定期更新，用于收录和整理个人制作的静态页面。
- 欢迎提交 Issue 或 Pull Request 建议改进，但请注意保持仓库的通用性和简洁性。
- 不承诺任何形式的持续更新或页面可用性。
- 所有页面内容仅供存档和展示用途。
